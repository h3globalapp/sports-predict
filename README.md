# Sports Predictor Pro

High-accuracy sports prediction Progressive Web App (PWA) powered by machine learning.

## Features

- ⚡ Real-time predictions for multiple sports
- 📊 Confidence scoring with visual indicators
- 🔥 Smart betting recommendations (STRONG/LEAN/NO BET)
- 💾 Offline support with cached predictions
- 📱 Installable on Android/iOS home screen
- 🔄 Auto-updates via service worker

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
